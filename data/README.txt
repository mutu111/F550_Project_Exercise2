Put historical price CSV files in this folder.

Default file path used by main.py:
data/AAPL.csv

Required columns:
Date, Close

Also supported:
Date, Adj Close

Example:
Date,Open,High,Low,Close,Volume
2022-01-03,177.83,182.88,177.71,182.01,104487900
2022-01-04,182.63,182.94,179.12,179.70,99310400
